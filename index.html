💡 HOW TO TYPE RUSSIAN ON YOUR KEYBOARD
The keys shown are for the standard Russian keyboard layout (Windows & macOS built-in). Go to Settings → Language → Add Russian, then switch with Win+Space (Windows) or Cmd+Space / Globe key (Mac). Keys map by position, not by sound — so v types М, d types В, etc.
